    위의 추천 시스템 파이썬 파일은 다음과 같은 입력을 사용하여 Recommender를 적용: 위에서 분류된 얼굴 모양, 사용자가 업도를 원하는지 여부 및 사용자 머리 길이.

 시스템은 얼굴 모양에 따라 6개의 헤어 스타일 이미지를 추천. 원래 선택은 무작위 숫자 점수를 기반으로 하지만 사용자는 그들의 최상위와 최하위 선호도를 시스템에 제공가능. 이 선호도는 점수를 추가하거나 제거함으로써 다음 사용자를 위한 우선 순위를 조정. 스타일이 마음에 든다면 다음 사용자를 위해 해당 스타일의 우선 순위가 높아지고, 사용자가 가장 좋아하지 않는 스타일은 5점을 잃게되어 사용자의 상위 6개 목록에서 벗어날 수도 있음

추천자 이미지는 모델을 훈련하는 데 사용한 이미지의 하위 집합입니다. 중복되는 헤어 스타일(특히, 동일한 인물의 이미지인 경우) 및 유행에 지나치게 뒤쳐지는 헤어스타일을 배제하기 위해 하위 집합으로 축소할 것!


## 작동 방식 순서 ##

    이미지 처리 및 데이터 프레임 생성: 이미지 디렉토리에서 헤어 스타일 이미지를 처리하고 데이터 프레임에 필요한 정보를 저장합니다.

    사용자 입력 받기: 사용자에게 얼굴 모양, 업두 여부 및 머리 길이와 같은 정보를 입력 받습니다.

    추천 스타일 표시: 사용자 입력에 기반하여 해당 얼굴 모양과 머리 길이에 맞는 추천 스타일을 선정하고 이를 시각적으로 표시합니다.

    선호도 반영: 사용자가 선호하는 스타일과 가장 선호하지 않는 스타일을 입력 받아, 추천 스타일의 점수를 조정합니다.

    추천 이미지 저장: 최종 추천 이미지를 저장하고 이를 반환합니다.
